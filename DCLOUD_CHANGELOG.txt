版本 1.0.2 (2025-11-07)

【首次发布到 DCloud 插件市场】

一、核心功能

1. VideoPlayer 视频播放器组件
   • 播放/暂停控制
   • 进度条拖拽调节
   • 快进/快退功能（可自定义步长，默认10秒）
   • 全屏播放支持（完美兼容 iOS/Android）
   • 自定义播放器图标
   • 视频播放事件监听

2. MediaGallery 媒体画廊组件
   • 视频分类展示和切换
   • 视频网格列表（支持自定义每页数量）
   • 分页功能
   • 二维码分享功能
   • 响应式布局

二、技术特性

• Vue 3 Composition API 开发
• 完整的 TypeScript 类型定义
• 响应式设计，完美适配 PC/平板/手机
• 无第三方依赖，轻量级（核心代码 < 50KB）
• 支持 ES Module 导入
• Tree-shaking 友好

三、平台兼容性

✓ H5
✓ App (iOS/Android)
✓ 微信小程序
✓ 支付宝小程序

四、使用场景

• 在线教育平台的视频课程展示
• 企业官网的产品视频展示
• 媒体资讯的视频新闻播放
• 短视频平台的内容管理
• 音视频网站的媒体库管理

五、文档和示例

• 完整的 API 文档
• Vue 3 项目集成示例
• TypeScript 类型文档
• 基础使用示例代码
