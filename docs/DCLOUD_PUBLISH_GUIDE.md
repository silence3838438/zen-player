## 🚀 DCloud 插件市场发布流程

### 步骤 1：注册账号

1. 访问 [DCloud 插件市场](https://ext.dcloud.net.cn/)
2. 点击右上角"登录/注册"
3. 使用手机号或邮箱注册账号
4. 完成实名认证（必需）

### 步骤 2：进入发布页面

1. 登录后，点击"发布插件"
2. 选择插件类型：**前端组件**
3. 填写基本信息

### 步骤 3：填写插件信息

#### 基本信息
```
插件名称：沉浸式音视频播放器
英文名称：zen-player
插件类型：前端组件
技术分类：Vue3
适用项目：Web 项目、Vue3 项目
开源协议：MIT
```

#### 插件简介（100字以内）
```
一款功能强大、界面精美的沉浸式音视频播放器插件。支持自定义播放控制、分类视频展示、全屏播放、响应式布局等功能。完美适配 PC/平板/手机端，提供 TypeScript 类型支持。适用于视频网站、在线教育、企业培训等场景。
```

#### 详细说明
将 README.md 的内容粘贴到详细说明中，包括：
- 功能特性
- 安装方法
- 使用示例
- API 文档
- 常见问题

#### 标签（关键词）
```
视频播放器, 音视频, Vue3, 播放器, media-player, video, 分类视频, 响应式, TypeScript
```

### 步骤 4：上传插件文件

#### 方式一：ZIP 压缩包上传
```bash
cd /Users/fgi/Desktop/zen-player-plugin
# 排除不必要的文件
zip -r zen-player-v1.0.0.zip . \
  -x "node_modules/*" \
  -x ".git/*" \
  -x ".DS_Store" \
  -x "*.log"
```

#### 方式二：Git 仓库导入
1. 将插件推送到 GitHub/Gitee
2. 在发布页面选择"从 Git 仓库导入"
3. 输入仓库地址
4. 选择分支和标签

**推荐使用 Git 仓库方式，便于后续更新**

### 步骤 5：上传截图和演示

1. **主图**（必需）：800x600，展示插件核心功能
2. **截图 1**：PC 端完整界面
3. **截图 2**：移动端响应式效果
4. **截图 3**：分类视频画廊
5. **截图 4**：视频播放控制面板
6. **演示 GIF**（可选）：展示操作流程

### 步骤 6：设置价格和授权

#### 免费版设置
```
价格类型：免费
下载限制：无
商用授权：允许（MIT License）
```

#### 付费版设置（未来）
```
价格类型：付费
个人版：¥199/年
企业版：¥999/年
试用期：7天
授权方式：按域名/按项目
```

### 步骤 7：提交审核

1. 检查所有信息是否填写完整
2. 预览插件页面效果
3. 同意《DCloud 插件发布协议》
4. 点击"提交审核"

**审核时间**：通常 1-3 个工作日
