# 截图指南

## 📸 需要的图片

### 1. 插件图标（icon.png）
- **尺寸**：128x128 px
- **格式**：PNG，透明背景
- **设计要素**：
  - 主元素：播放按钮 ▶️
  - 辅助元素：音视频波形或媒体符号
  - 背景：渐变色（建议：#4158D0 → #C850C0）
  - 风格：现代、扁平化、简洁

### 2. 预览图（建议尺寸：750x1334 px 或 1920x1080 px）

#### 图1：主界面全览（preview-1-main.png）
**截图内容：**
- 完整的媒体画廊界面
- 显示分类标签
- 显示视频网格列表
- 展示整体布局

**截图步骤：**
1. 运行 powerfulPlayer 项目
```bash
cd /Users/fgi/Desktop/powerfulPlayer
npm run dev
```
2. 浏览器访问 http://localhost:5173
3. 截取完整页面（包含头部、分类、视频列表）
4. 调整窗口大小为合适比例
5. 使用截图工具保存

#### 图2：视频播放器（preview-2-player.png）
**截图内容：**
- 视频播放器界面
- 显示播放控制按钮
- 显示进度条
- 显示视频信息

**截图步骤：**
1. 点击任意视频开始播放
2. 等待视频加载完成
3. 暂停在有内容的画面
4. 确保控制栏显示
5. 截取播放器区域

#### 图3：分类切换（preview-3-categories.png）
**截图内容：**
- 多个分类标签
- 分类切换效果
- 不同分类的视频列表

**截图步骤：**
1. 点击不同的分类标签
2. 展示分类切换后的效果
3. 可以使用浏览器开发工具调整为移动端视图
4. 截取包含分类和列表的区域

#### 图4：响应式设计（preview-4-responsive.png）
**截图内容：**
- PC端和移动端对比
- 或者单独展示移动端效果

**截图步骤：**
1. 打开浏览器开发工具（F12）
2. 切换到设备模拟器
3. 选择 iPhone 或 Android 设备
4. 截取移动端界面
5. 可以用拼图工具将PC端和移动端放在一起

#### 图5：二维码分享（preview-5-qrcode.png）（可选）
**截图内容：**
- 显示二维码功能
- 展示分享界面

**截图步骤：**
1. 确保二维码功能开启
2. 截取包含二维码的界面
3. 可以添加说明文字

## 🛠️ 截图工具推荐

### macOS
- **自带截图**：Cmd + Shift + 4
- **CleanShot X**：专业截图工具（付费）
- **Snipaste**：免费，功能强大

### 截图美化工具
- **Screely**（screely.com）- 在线美化截图
- **Carbon**（carbon.now.sh）- 代码截图
- **Shots**（shots.so）- 设备模型截图

## 📐 图片规格要求

### DCloud 插件市场要求
- **图标**：128x128 px，PNG格式
- **预览图**：
  - 最小尺寸：375x667 px
  - 推荐尺寸：750x1334 px（2倍图）
  - 最大尺寸：1920x1080 px
  - 格式：PNG 或 JPG
  - 数量：3-5张

### 图片命名规范
```
screenshots/
├── icon.png                    # 插件图标 128x128
├── preview-1-main.png         # 主界面
├── preview-2-player.png       # 播放器
├── preview-3-categories.png   # 分类切换
├── preview-4-responsive.png   # 响应式（可选）
└── preview-5-qrcode.png       # 二维码（可选）
```

## 🎨 设计风格建议

### 配色方案
- **主色调**：#4158D0（蓝紫色）
- **辅助色**：#C850C0（粉紫色）
- **背景色**：#f0f8ec（浅绿色）
- **文字色**：#333333（深灰）

### 设计原则
1. **简洁明了**：突出核心功能
2. **视觉统一**：保持风格一致
3. **清晰可读**：文字和按钮清晰
4. **真实展示**：实际功能截图，不夸大

## 🚀 快速截图流程

### 一键截图脚本（需要先运行项目）

1. **启动项目**
```bash
cd /Users/fgi/Desktop/powerfulPlayer
npm run dev
```

2. **浏览器访问**
```
http://localhost:5173
```

3. **截图检查清单**
- [ ] 主界面全览（显示所有功能）
- [ ] 视频播放器（播放状态）
- [ ] 分类切换（展示分类功能）
- [ ] 响应式设计（PC或移动端）
- [ ] 特色功能（二维码等）

4. **图片处理**
- 使用 Screely 或类似工具美化截图
- 添加圆角、阴影等效果
- 调整为统一尺寸
- 保存到 screenshots 目录

## 💡 临时方案

如果暂时没有设计资源，可以：

1. **使用占位图**
   - 先用文字说明上传到 DCloud
   - 审核通过后再更新图片

2. **简单设计**
   - 使用纯色背景 + 文字
   - 图标：蓝色圆形 + 白色播放按钮
   - 预览：直接截图不美化

3. **后期更新**
   - DCloud 支持版本更新时修改图片
   - 可以先上传简单版本
   - 后续优化设计后再更新
